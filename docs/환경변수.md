# 환경변수

실제 값은 `.env`에 넣고 **절대 커밋하지 않는다.** 예시는 [`.env.example`](../.env.example) 참고.

## 현재 사용 중
- (없음) — 스켈레톤 단계라 실제 환경변수를 사용하지 않는다.

## 향후 추가 예정 (백엔드/AI 연결 시)
| 이름 | 용도 | 노출 범위 |
| --- | --- | --- |
| `ANTHROPIC_API_KEY` | AI 요약/분석 | 서버 전용(클라이언트 금지) |
| `BACKEND_API_URL` | 본문 추출 백엔드 주소 | `EXPO_PUBLIC_` 접두사로 클라이언트 노출 가능 |

## Source Ingestion Toolkit (future · 전부 서버 전용)
수집 백엔드가 생길 때 사용할 변수. 상세 정책: `docs/구현로그/2026-07-03_source_ingestion_toolkit_reference.md`.

| 이름 | 용도 | 기본값 / 정책 |
| --- | --- | --- |
| `FISH_AUDIO_API_KEY` | **Fish Audio TTS** (audioScript→음성 생성) | **서버 전용 · 클라이언트(앱) 노출 절대 금지.** `.env.example`엔 이름만 |
| `GROQ_API_KEY` | Whisper 음성→텍스트(자막 없는 영상) | 미설정 시 비활성. 서버 전용 |
| `YOUTUBE_API_KEY` | (future) YouTube 메타/검색 | 선택. 미설정 시 비활성 |
| `REDDIT_CLIENT_ID` / `REDDIT_CLIENT_SECRET` | (future) Reddit API | 선택. API 정책 검토 후 |
| `X_BEARER_TOKEN` | (future) X 수집 | 선택. X는 high risk / future |
| `OPENAI_API_KEY` | Whisper 폴백 / AI | 서버 전용. 클라이언트 금지 |
| `EXA_API_KEY` | (선택) 시맨틱 검색 | 미설정 시 비활성 |
| `GITHUB_TOKEN` | GitHub 수집 레이트리밋 | 선택 |
| `INGEST_ALLOW_COOKIES` | 쿠키 기반 수집 허용 플래그 | **기본 `false`** (민감·운영 자동화 금지) |
| `INGEST_ALLOW_INSANE_SEARCH` | WAF fallback 허용 플래그 | **기본 `false`** (공개 페이지 + 별도 검증 후에만) |
| `INGEST_ALLOW_LOGIN_SCRAPE` | 로그인 기반 수집 허용 | **기본 `false`** (운영 금지) |

- **민감 기능 3종(쿠키·insane-search·로그인)은 플래그로 기본 비활성.** 운영 서버에서 켜지 않는다.
- 쿠키/토큰 등 자격증명은 서버에만 두고 **절대 커밋 금지**. 원문은 저장하지 않거나 temporary cache만.

## 규칙
- 클라이언트에 노출할 값만 `EXPO_PUBLIC_` 접두사를 붙인다.
- 비밀 키(AI/DB/service role 등)는 클라이언트에 넣지 않는다.
