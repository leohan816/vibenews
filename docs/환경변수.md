# 환경변수

실제 값은 서버의 `mode-600 .env.server.local`(리포에 커밋 금지)과 디바이스 SecureStore에만 둔다.
리포에는 **이름/자리표시자만** 담은 [`.env.example`](../.env.example)만 커밋한다. `.gitignore`가 `.env.*`를
막고 `!.env.example`만 예외로 둔다.

## 서버 런타임 (구현 정의 · 시작 전 필요)
| 이름 | 용도 | 노출 범위 |
| --- | --- | --- |
| `VIBENEWS_ENV` | 실행 환경 식별자 | 서버 전용 |
| `VIBENEWS_BIND_HOST` | API 바인드 주소(항상 loopback `127.0.0.1`) | 서버 전용 |
| `VIBENEWS_PORT` | API 포트 | 서버 전용 |
| `VIBENEWS_STATE_DIR` | DB/오디오/스테이징 상태 디렉터리 | 서버 전용 |
| `VIBENEWS_USER_TIMEZONE` | 일일 TTS 상한 리셋 기준 타임존(`Asia/Seoul`) | 서버 전용 |
| `VIBENEWS_DEVICE_TOKEN_SHA256` | 인가된 디바이스 bearer 토큰의 SHA-256 | 서버 전용 |
| `YTDLP_BINARY` | 자막 취득에 쓰는 yt-dlp 실행 파일 경로 | 서버 전용 |

## D-003 승인 provider 키 (이름만 커밋 · 값은 undisclosed)
프로즌 설계가 허용하는 provider는 **DeepSeek Builder/Verifier와 Fish TTS 둘뿐**이다. 대체/로컬 LLM,
provider fallback은 금지되며 `config.ts`가 비허용 키 접두사(`OPENAI_/ANTHROPIC_/KIMI_/QWEN_`)를 **거부**한다.

| 이름 | 용도 | 노출 범위 |
| --- | --- | --- |
| `DEEPSEEK_API_KEY` | DeepSeek 인증(요청 헤더 전용, 본문에 절대 미포함) | 서버 전용 |
| `DEEPSEEK_BASE_URL` | DeepSeek 엔드포인트 | 서버 전용 |
| `DEEPSEEK_BUILDER_MODEL` | Builder 모델 선택자(`reasoning_effort` 미전송) | 서버 전용 |
| `DEEPSEEK_VERIFIER_MODEL` | 별도 Verifier 모델 선택자 | 서버 전용 |
| `DEEPSEEK_VERIFIER_REASONING_EFFORT` | Verifier 전용 reasoning effort | 서버 전용 |
| `FISH_API_KEY` | Fish TTS 인증(헤더 전용) | 서버 전용 |
| `FISH_TTS_MODEL` | Fish 모델 선택자 | 서버 전용 |
| `FISH_REFERENCE_ID` | 승인된 음성 reference id | 서버 전용 |

## 클라이언트 nonsecret
| 이름 | 용도 | 노출 범위 |
| --- | --- | --- |
| `EXPO_PUBLIC_VIBENEWS_API_BASE_URL` | tailnet 전용 Tailscale Serve HTTPS 주소 | Expo가 클라이언트에 인라인 |

## 규칙
- 클라이언트에 노출할 값만 `EXPO_PUBLIC_` 접두사를 붙인다. 디바이스 bearer 토큰은 SecureStore에만 둔다.
- 비밀 키(provider/토큰)는 클라이언트에 넣지 않고 서버 `.env.server.local`에만 둔다.
- `.env.server.local`은 열람/출력/복사/커밋 금지. provider 실제 값은 로그/리포트/픽스처에 남기지 않는다.
