# 실행 방법

## 준비
```bash
npm install
```

## 개발 서버
```bash
npm run web       # 웹 브라우저 (가장 빠른 확인)
npm run ios       # iOS 시뮬레이터 (macOS 필요)
npm run android   # Android 에뮬레이터
npm start         # Expo Dev 서버 (QR로 실기기 Expo Go)
```

## 참고
- 스택: Expo (React Native) + Expo Router. 자세한 방향은 [`설계문서/00_제품_전체지도.md`](../설계문서/00_제품_전체지도.md).
- 실제 실행 검증 결과/스크린샷은 구현 커밋 이후 [`구현로그/`](구현로그/)에 기록한다.
