# 테스트 방법

## 타입/린트
```bash
npx tsc --noEmit   # 타입 체크
npm run lint       # expo lint
```

## 수동 확인 (스켈레톤 단계)
- `npm run web` 실행 후 하단 탭 5개(Listen/Briefing/Recap/Saved/Settings) 이동 확인.
- Listen 중앙 재생 버튼 → 재생 화면 이동 확인.
- 각 화면의 mock/준비 중 배지 표시 확인.

## 참고
- 자동화 테스트는 아직 없음. 실제 기능 연결 시 블록별로 검증 기준을 설계문서의 "검증 방법" 항목에 맞춰 추가한다.
- 검증 결과는 [`구현로그/`](구현로그/)에 남긴다.
