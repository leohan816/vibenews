# Quality Notes — 01_ai_agent_video (synthetic)

> 같은 영상(예시)을 두고 **좋은 브리핑 vs 나쁜 브리핑**의 차이를 기록한다. 이 노트는 향후 prompt
> regression test의 채점 기준으로 쓴다.

## ❌ 나쁜 요약 (실패 예)

> "이 영상은 AI 에이전트에 대한 중요한 영상입니다. 에이전트가 인터넷을 사용할 수 있게 되었고, 이는
> 매우 중요합니다. 여러 플랫폼을 지원합니다. 앞으로 많이 쓰일 것입니다."

**왜 나쁜가**

- "중요합니다"만 반복 — 핵심 주장이 없다.
- self-healing routing(핵심 개념)이 사라졌다.
- 예시·도구명·숫자(yt-dlp, RSS, 14개 채널)가 모두 빠졌다.
- 모든 내용을 균등하게 뭉갰다(설치 잡담과 핵심을 같은 무게로).
- 진행자 **의견**("우회가 진짜 가치")을 사실처럼 말함.
- **예측**("많이 쓰일 것")을 확정처럼 말함.
- "그래서 뭘 해야 하지?"가 없다.

## ✅ 좋은 요약 (기준)

[`standard_brief_7min.example.md`](standard_brief_7min.example.md) 참고.

**왜 좋은가**

- coreThesis("접근이 아니라 우회")가 살아 있다.
- 왜 중요한지(학습 컷오프 문제) 설명한다.
- 근거·예시·도구명·숫자를 보존한다.
- 원문 시간 순서(설치 데모 포함)가 아니라 **문제 → 개념 → 작동** 논리로 재구성한다.
- **사실/의견/예측/불확실**을 구분한다.
- VibeNews와 실제로 맞닿는 지점만 연결(억지 X).
- 행동("자동화 쉬운 채널부터")과 다음 질문을 준다.

## 채점 축 (BriefingQualityScore)

| 축                | 나쁜 요약            | 좋은 요약       |
| ----------------- | -------------------- | --------------- |
| thesisAccuracy    | 낮음(주장 소실)      | 높음            |
| evidenceRetention | 낮음(예시·숫자 소실) | 높음            |
| specificity       | 낮음(일반론)         | 높음            |
| cautionBalance    | 낮음(의견=사실)      | 높음            |
| userRelevance     | 없음                 | 있음(억지 아님) |

## 검수(verifier)가 잡아야 할 것

- 빠진 핵심: self-healing routing
- 왜곡: 의견을 사실로
- 과장: 예측을 확정으로
- numeric/entity check: "14개+ 채널", 도구명(yt-dlp 등) 정확한가

## 요약을 그대로 읽으면 왜 나쁜가

- AnalyticSummary는 **정보 정리용**이라 문장이 딱딱하다. 그대로 TTS에 넣으면 로봇이 목록을 읽는
  느낌.
- 오프닝·전환·말맛이 없어 **듣다가 이탈**한다.
- "핵심 주장: … / 근거: …" 같은 라벨식 문장은 눈으로 읽는 문서지 귀로 듣는 낭독문이 아니다.
- 그래서 VibeNews는 요약 → **음성용 rewrite(SpokenAudioScript)** → 검수 → TTS 순서를 강제한다.

## 좋은 음성 스크립트(SpokenAudioScript)의 특징

- 자연스러운 오프닝으로 시작한다("오늘 이 영상, 핵심은 하나예요").
- 문장이 이어지고 **전환문**이 있다("그런데 진짜 포인트는…").
- 숫자·도구명은 살리되 **말하듯** 녹인다.
- 과장 없이 주의점을 자연스럽게 끼운다("이건 기대일 뿐 확정은 아니에요").
- 관심사 연결이 억지스럽지 않다.
- 마지막에 행동 한 가지를 남긴다.

## robotic phrasing vs human-read phrasing

| robotic (❌)                           | human-read (✅)                                     |
| -------------------------------------- | --------------------------------------------------- |
| "핵심 주장은 다음과 같습니다. 첫째, …" | "핵심은 딱 하나예요 — …"                            |
| "근거: 채널 차단 시 재시도."           | "한 통로가 막히면 알아서 다른 길로 돌아가더라고요." |
| "결론적으로 매우 중요합니다."          | "그래서 Leo님 프로젝트엔 이렇게 쓸 수 있어요."      |
| "본 영상은 AI 에이전트를 다룹니다."    | "오늘 이 30분 영상, 같이 뼈대만 추려볼게요."        |
